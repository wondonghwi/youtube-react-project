## YouTube Viewer: React & TypeScript 기반 유튜브뷰어

**[링크](https://youtube-react-ts.netlify.app/)**

### 🛠 기술 스택

- React
- TypeScript: JavaScript에 타입을 추가하여 더 안정적인 코드를 작성
- YouTube Data API: 유튜브 비디오 및 사용자 데이터에 액세스

### ✨ 주요 기능

- 비디오 검색 및 재생
- 최신 트렌드 비디오 목록 보기
- 사용자 재생 목록 관리
